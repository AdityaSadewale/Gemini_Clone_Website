# 🚀 Gemini Clone Website


A modern **Gemini AI Clone Web Application** built using React and the Google Gemini API. This project replicates the core functionality of Google's Gemini AI, providing a clean and interactive chat interface for users.

---

## 📌 Overview

Gemini Clone Website is a responsive AI-powered chatbot that allows users to interact with an intelligent conversational model. It mimics the behavior of Google Gemini, delivering real-time responses with a smooth and intuitive UI.

This project is ideal for learning:

* API integration
* React-based UI development
* Building AI chat interfaces

## ⚙️ Installation & Setup

Follow these steps to run the project locally:

### 1️⃣ Clone the repository

```bash
git clone https://github.com/AdityaSadewale/Gemini_Clone_Website.git
```

### 2️⃣ Navigate to the project folder

```bash
cd Gemini_Clone_Website
```

### 3️⃣ Install dependencies

```bash
npm install
```

### 4️⃣ Add environment variables

Create a `.env` file in the root directory and add:

```env
VITE_GEMINI_API_KEY=your_api_key_here
```

### 5️⃣ Run the development server

```bash
npm run dev
```
